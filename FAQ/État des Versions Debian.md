# État des Versions Debian (Mai 2026)

Ce document récapitule les versions de Debian GNU/Linux selon la nomenclature officielle du projet.

## 📊 Tableau des Versions


| Statut | Version | Nom de code | Sortie Initiale | Fin de Support |
| :--- | :--- | :--- | :--- | :--- |
| **Dernière Stable** | **13** | **Trixie** | Août 2025 | ~Août 2030 (LTS inclus) |
| **Avant-dernière (Oldstable)** | **12** | **Bookworm** | Juin 2023 | Juin 2028 (LTS inclus) |
| **Oldoldstable** | **11** | **Bullseye** | Août 2021 | Août 2026 (Fin LTS imminente) |

## ✅ Preuves de la hiérarchie
1. **Cycle de Publication :** Debian publie une nouvelle version stable environ tous les 2 ans. La version 13 (Trixie) est l'actuelle "Stable".
2. **Cycle de Vie :** Dès qu'une nouvelle version sort, la précédente devient automatiquement "Oldstable". Bookworm est donc l'avant-dernière version stable.
3. **État de maintenance :** La version 12.x continue de recevoir des correctifs de sécurité critiques via l'équipe Debian Security ou l'équipe LTS.
